# Campus Course Records Manager (CCRM)

## Overview
CCRM is a Java console-based application to manage student records, courses, enrollments, grades, transcripts, and file operations.

## Features
- Add, update, deactivate students and courses
- Manage student course enrollments with credit limits
- Grade calculation and GPA computation
- Import/export CSV files for students and courses
- Backup data directory recursively
- Console menu-driven interface

## Technologies Used
- Java SE 11+
- Java Streams, Lambdas, NIO.2
- Singleton and Builder Design Patterns
- Custom Exception Handling
